# finanzleser.de Frontend

Next.js 15 Frontend für finanzleser.de – Deutsches Finanzmagazin von Finconext GmbH.

**Stack:** Next.js 15 · TypeScript · Tailwind CSS v4 · GSAP · Framer Motion · WPGraphQL

**Lokale Entwicklung:**
```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.
